# Job Market & Skill Demand Trends Analyzer

## Overview
This project analyzes job market trends and identifies the most in-demand skills using real-world job posting data.

The project uses **Python, NLP techniques, Excel, and Power BI** to extract insights from job descriptions and visualize hiring patterns.

## Tools Used
- Python
- spaCy (NLP)
- Excel
- Power BI
- Kaggle Dataset – LinkedIn Job Postings

## Key Insights
- Most demanded job skills
- Hiring trends by location
- Salary trends across experience levels
- Companies with highest skill demand
